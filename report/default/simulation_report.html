Simulation settings

Path and folder

VariableValueDescription
OutputPath[%outputpath%]Path to output folder
MeteoPath[%meteopath%]Path to meteorological stations folder
PhenoPath[%phenopath%]Path to phenological parameters folder
IrrMethPath[%irrmethpath%]Path to irrigation methods folder
IrrMethFileName[%irrmethfilename%]File, located in irrigation methods folder, in which irrigation methods filenames are stored
WatSourPath[%watsourpath%]Path to water sources folder

Simulation mode

VariableValueDescription
Mode[%mode%]Type of simulation

0: simulation without irrigation;
1: simulation with irrigation, mode consumptions;
2: simulation with irrigation, field capacity needs satisfaction;
3: simulation with irrigation, fixed volumes;
4: fixed irrigation applications, data and volumes are specified in a file.

Simulation parameters

VariableValueDescription
CapillaryFlag[%capillaryflag%]Simulate capillary rise (True/False)
MeteoStatWeightNum[%meteostatweightnum%]Number of nearest meteorological stations used to weight crop phenology
StartIrrSeason[%startirrseason%]Start irrigation at day [1...366]
EndIrrSeason[%endirrseason%]Stop irrigation season at day [1...366]
zEvap[%zevap%]Evaporative layer depth (m)
zRoot[%zroot%]Transpirative layer depth (m)

Periodical output setting

VariableValueDescription
MonthlyFlag[%monthlyflag%]Export outputs aggregated by months (True/False)
If MonthlyFlag is False, the following settings are used:
StartDate[%startdate%]Save output starting from day [1...366]
EndDate[%enddate%]Stop output at day [1...366]
DeltaDate[%deltadate%]Save outputs every n days [1...366]
